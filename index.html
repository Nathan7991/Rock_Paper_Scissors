Rock, Paper, Scissors

Hey, thanks for stopping by my page.


Here's how you can play the game:

  1. Start by activating the Chrome Developer Tools however you want. (I like the shift+ctrl+j shortcut)
  2. Once you're there, enter/copy + paste the following: game();
  3. game(); is what is called a function. When you enter this function into the console, you are also activating code written inside of the function declaration to run. Inside of the function are the key code needed to play the game!

I hope you enjoy it!
